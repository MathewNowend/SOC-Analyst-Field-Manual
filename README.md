# SOC Analyst Field Manual

> A practical cybersecurity reference for SOC Analysts, Blue Team defenders, Detection Engineers, Incident Responders, and students entering defensive security.

---

## About This Project

The SOC Analyst Field Manual is an original open-source knowledge base designed to document practical investigation workflows used by modern Security Operations Centers.

Rather than focusing solely on theory, this project emphasizes:

- Incident investigation methodology
- Detection engineering
- Threat hunting
- Log analysis
- MITRE ATT&CK mapping
- Windows Event IDs
- Microsoft Defender XDR
- Microsoft Sentinel (KQL)
- Splunk SPL
- Sigma Rules
- PowerShell
- Linux investigations
- Active Directory security
- Cloud security investigations

Every guide is written from a defender's perspective and is intended to provide repeatable investigation processes that can be adapted to real enterprise environments.
